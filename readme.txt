== Description ==

**SpamGuard AI** es un plugin anti-spam revolucionario que utiliza Machine Learning para proteger tu sitio WordPress de comentarios spam.

= Características principales =

* **Inteligencia Artificial**: Utiliza modelos de ML entrenados específicamente para detectar spam
* **Aprendizaje Continuo**: Mejora automáticamente con cada corrección que haces
* **Alta Precisión**: Detecta spam que otros plugins no pueden identificar
* **Fácil de usar**: Configuración en menos de 1 minuto
* **Sin falsas alarmas**: Sistema inteligente que minimiza falsos positivos
* **Dashboard completo**: Estadísticas detalladas y logs de análisis
* **Privacidad**: Tu contenido no se comparte públicamente

= ¿Cómo funciona? =

1. Instala el plugin
2. Genera tu API Key gratuita
3. ¡Listo! El plugin comenzará a proteger tu sitio automáticamente

Cada comentario es analizado en tiempo real antes de ser publicado. El sistema aprende de tus correcciones para mejorar continuamente.

= Ventajas sobre otros plugins anti-spam =

* **Más inteligente**: ML vs reglas simples
* **Se adapta a tu sitio**: Aprende del spam específico que recibes
* **Sin CAPTCHA**: No molesta a tus visitantes legítimos
* **Transparente**: Sabes exactamente por qué un comentario fue marcado

== Installation ==

= Instalación automática =

1. Ve a Plugins > Añadir nuevo
2. Busca "SpamGuard AI"
3. Haz clic en "Instalar ahora"
4. Activa el plugin
5. Ve a SpamGuard AI > Configuración
6. Haz clic en "Generar API Key Automáticamente"

= Instalación manual =

1. Descarga el plugin
2. Sube la carpeta `spamguard` a `/wp-content/plugins/`
3. Activa el plugin desde el menú Plugins
4. Configura tu API Key en Configuración

== Frequently Asked Questions ==

= ¿Es gratis? =

Sí, el plugin es completamente gratuito. La API tiene un tier gratuito generoso que es suficiente para la mayoría de sitios.

= ¿Necesito conocimientos técnicos? =

No. La configuración toma menos de 1 minuto con el botón de registro automático.

= ¿Funciona con otros plugins anti-spam? =

Sí, pero recomendamos desactivar otros plugins anti-spam para evitar conflictos.

= ¿Qué pasa si el plugin se equivoca? =

Simplemente marca/desmarca el comentario como spam desde WordPress. El sistema aprenderá de tu corrección.

= ¿Los datos son privados? =

Sí. Solo se envían metadatos necesarios para el análisis (longitud del texto, cantidad de enlaces, etc.). El contenido completo permanece en tu servidor.
